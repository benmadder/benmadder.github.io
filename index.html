My Projects

Solar System Generator

In this project, I designed a web-based program through the use of p5.js that allows the user to create a simple galaxy simulation that meets their desired specifications. This project allowed me to further develop my ability to translate my ideas into actual usable creations through programming.

Watch Drop

Watch Drop is an exotic marketplace that utilised a dynamic website through an SQLite database to display changing information such as watch prices, bids, listings and allow for the creation of user accounts. This project provided me with the opportunity to evolve my web developing skills by allowing me to focus on dynamic functionality through the utilisation of the Bootstrap framework, Python Flask framework, and the use of SQLAlchemy.

Harmony Lamp

A lamp designed to provide people with the ability to create tangible connections with one another, even when separated due to circumstances such as the COVID-19 pandemic. With this prototype I refined my ability with Arduino programing and utilised research into the field of tangible media and use of users physical interactions to create meaningful user experiences.

Escape Travel Blog

Escape is a travel blog with a focus on northern European countries, here potential tourists can get an idea of the given destinations and what attractions they can expect. This project was one of my initial explorations into web development where I learned how to use the languages of HTML and CSS along with several important design principles which allowed me to create aesthetically pleasing, organised and user-centric websites.
